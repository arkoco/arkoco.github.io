OpenClaw 操作指南

📋 概述​

OpenClaw 是一个通过 npm 安装的自动化工具包，用于管理消息通道、AI代理和节点服务。

当前版本： openclaw@2026.2.25 安装方式： npm 全局安装 运行环境： Node.js v24.13.0

🔍 当前运行状态​

进程列表​

# 查看所有 OpenClaw 进程
ps aux | grep openclaw

# 当前运行的进程：
# - openclaw (主进程，PID 1301404)
# - openclaw-gateway (WebSocket 网关，PID 1301413)
# - openclaw (当前会话，PID 1360885)

网关状态​

# 查看网关状态
openclaw gateway status

当前配置：

  • 监听地址：127.0.0.1:18789
  • 绑定模式：loopback (仅本地访问)
  • 日志文件：/tmp/openclaw/openclaw-2026-02-27.log
  • 配置文件：~/.openclaw/openclaw.json
  • 服务状态：systemd (禁用)

控制面板：

  • Dashboard: http://127.0.0.1:18789/

🚀 启动与停止​

方式一：前台运行（推荐用于开发/测试）​

# 启动网关（前台运行）
openclaw gateway run

# 或者直接运行完整服务
openclaw

方式二：systemd 服务（推荐用于生产环境）​

# 安装网关服务
openclaw gateway install

# 启动服务
openclaw gateway start

# 停止服务
openclaw gateway stop

# 重启服务
openclaw gateway restart

# 查看服务状态
openclaw gateway status

# 卸载服务
openclaw gateway uninstall

方式三：使用 Node.js 脚本​

# 使用 npm 脚本启动
npm start

# 或者直接运行 CLI 入口点
node ~/.nvm/versions/node/v24.13.0/lib/node_modules/openclaw/openclaw.mjs

🛠️ 常用命令​

网关管理​

# 查看网关健康状态
openclaw gateway health

# 查看网关发现（本地和远程）
openclaw gateway discover

# 调用网关 RPC 方法
openclaw gateway call health

# 查看网关使用成本
openclaw gateway usage-cost

系统管理​

# 查看完整状态
openclaw status

# 运行健康检查
openclaw doctor

# 查看帮助
openclaw --help

# 查看网关帮助
openclaw gateway --help

# 查看版本
openclaw --version

会话管理​

# 列出所有会话
openclaw sessions list

# 查看会话历史
openclaw sessions history --session-key <key>

# 发送消息到会话
openclaw sessions send --session-key <key> --message "Hello"

通道管理​

# 列出已连接的通道
openclaw channels list

# 配置通道
openclaw configure

# 查看通道帮助
openclaw channels --help

设备管理​

# 列出配对设备
openclaw devices list

# 配对新设备
openclaw pairing

# 生成 iOS 配对二维码
openclaw qr

模型管理​

# 发现并配置模型
openclaw models

系统监控​

# 查看系统事件
openclaw system

# 查看健康状态
openclaw health

# 查看心跳状态
openclaw heartbeat

📦 安装与更新​

安装 OpenClaw​

# 全局安装最新版本
npm install -g openclaw

# 安装特定版本
npm install -g openclaw@2026.2.25

更新 OpenClaw​

# 更新到最新版本
npm update -g openclaw

# 检查更新状态
openclaw update

卸载 OpenClaw​

# 卸载 CLI（保留本地配置和数据）
openclaw uninstall

# 完全卸载（包括数据和配置）
npm uninstall -g openclaw

⚙️ 配置文件​

配置位置​

  • CLI 配置： ~/.openclaw/openclaw.json
  • 状态目录： ~/.openclaw/
  • 日志目录： /tmp/openclaw/

配置命令​

# 交互式配置向导
openclaw configure

# 获取配置
openclaw config get <key>

# 设置配置
openclaw config set <key> <value>

# 删除配置
openclaw config unset <key>

🔧 故障排查​

常见问题​

1. 端口被占用​

# 查看端口占用
lsof -i :18789

# 强制启动（杀掉占用进程）
openclaw gateway --force

2. 服务未安装​

# 安装服务
openclaw gateway install

# 启用并启动服务
systemctl enable openclaw-gateway.service
systemctl start openclaw-gateway.service

3. 权限问题​

# 使用 sudo 运行（如果需要）
sudo openclaw gateway start

# 检查权限
ls -la ~/.openclaw/

4. 配置错误​

# 运行诊断
openclaw doctor

# 修复问题
openclaw doctor --repair

查看日志​

# 查看网关日志
openclaw logs

# 查看特定日志文件
tail -f /tmp/openclaw/openclaw-2026-02-27.log

# 查看系统日志
journalctl -u openclaw-gateway -f

🔄 服务重启步骤​

生产环境重启​

# 1. 停止服务
openclaw gateway stop

# 2. 验证进程已停止
ps aux | grep openclaw

# 3. 启动服务
openclaw gateway start

# 4. 检查状态
openclaw gateway status

快速重启（前台）​

# 停止当前进程
pkill -f "openclaw"

# 重新启动
openclaw gateway run

📊 监控命令速查​

命令功能
openclaw status查看通道健康和会话
openclaw gateway status查看网关服务状态
openclaw gateway health查看网关健康
openclaw gateway discover发现本地/远程网关
openclaw doctor健康检查和修复
openclaw logs实时查看日志
openclaw sessions list列出所有会话

🎯 快速开始​

# 1. 安装 OpenClaw
npm install -g openclaw

# 2. 配置
openclaw configure

# 3. 启动网关
openclaw gateway run

# 4. 打开控制面板
open http://127.0.0.1:18789/

# 5. 开始使用
openclaw --help
